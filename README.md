# 介绍

[staticfile.org](http://staticfile.org)命令行工具

![](https://api.travis-ci.org/staticfile/cli.svg)

# 安装

```
npm install -g sfile
```

# 使用

## 搜索

搜索关键字

```
sfile search [library]
```

## 获取链接

获取一个库的所有文件链接

```
sfile get [library]

Options:
  -s, --ssl      Output HTTPS link
```

## 查看版本

```
sfile --version
```
